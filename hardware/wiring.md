# Подключение компонентов

## Плата STM32

Целевая плата: STM32F401CCU6 Black Pill.

## Датчик VL53L0X

| Пин VL53L0X | Пин STM32 |
|---|---|
| VIN | 3V3 |
| GND | GND |
| SCL | PB6 |
| SDA | PB7 |
| GPIO1 | не подключён |
| XSHUT | не подключён |

Датчик работает по шине I2C1. Для базовой версии проекта используются только питание и линии I2C.

## Модуль microSD

| Пин microSD-модуля | Пин STM32 |
|---|---|
| VCC | 5V |
| GND | GND |
| CS | PA4 |
| SCK | PA5 |
| MISO | PA6 |
| MOSI | PA7 |

Модуль microSD работает по SPI1. Линия `CS` управляется вручную через GPIO `PA4`.

В используемом модуле питание `VCC` подключается к 5V, потому что на модуле есть собственная обвязка питания для карты памяти. Если используется другой модуль с маркировкой `3V3`, его нельзя подключать к 5V без проверки схемы модуля.

## Кнопки

| Кнопка | Назначение |
|---|---|
| KEY | запуск сессии подсчёта |
| NRST | перезапуск микроконтроллера |
| BOOT0 | вход в режим USB DFU для прошивки |

Кнопка KEY настроена как GPIO Input с подтяжкой Pull-up. Нажатое состояние соответствует `GPIO_PIN_RESET`.

## Индикация

Синий LED используется как индикатор состояния `BLOCKED`. Когда объект находится в контролируемой зоне, LED горит. Когда зона свободна, LED выключен.
